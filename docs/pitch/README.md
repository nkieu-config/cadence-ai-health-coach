- deck ตอบเกณฑ์ครบ 9 ข้อตามตาราง mapping ใน docs/10
- ⚠️ **รัน `npm run refresh:demo-week` ในสัปดาห์ที่ pitch (คำสั่งเดียวจบ)** — อุ่นทั้ง 2 อย่างที่เน่ารายวัน:

  - ไม่แตะ check-in/แชท · ต่างจาก `npm run seed` ที่ล้างทุกอย่าง
  - อยากอัปเดตแค่ goal ไม่ยิง Gemini: `npm run refresh:demo-week -- --goal-only`
