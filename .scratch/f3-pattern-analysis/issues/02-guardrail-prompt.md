### ⚠️ กับดักตอนแก้ที่เกือบพลาด

`findForbiddenTerms()` ใช้ `includes()` → ถ้าใส่คำว่า **`"อด"`** เดี่ยว ๆ ลงรายการ จะพังทันที เพราะมันอยู่ใน **"ตลอด" "ปลอดภัย" "ยอด" "อดทน"**
→ เป้าหมายดี ๆ อย่าง "เดินเล่น**ตลอด**ทางกลับบ้าน" จะถูกปฏิเสธ
จึงใส่เป็นวลีเต็ม (`อดอาหาร` `อดข้าว` `อดมื้อ` …) + **เขียนเทสต์คุมทั้ง 2 ด้าน** (ต้องปฏิเสธของแย่ / **ต้องไม่ปฏิเสธของดี**)

### 🚨 เจอความเสี่ยงวัน pitch แบบไม่ได้ตั้งใจหา → INFRA-07

ยิงจนชนเพดานจริง: **Gemini free tier = 20 ครั้ง/วัน/โปรเจกต์** (ตัวเลขจาก error ของ API ไม่ใช่เดา)
prod ใช้ key เดียว → **ทั้งแอปมีโควตา 20 ครั้ง/วัน** · แชท 1 ข้อความ = 1 ครั้ง
**cache ใน `ai_outputs` ไม่ใช่ optimization อีกต่อไป — มันคือสิ่งเดียวที่ทำให้แอปใช้งานได้จริง**
