## งาน

- [x] นอน: กราฟแท่งชั่วโมงนอนรายวัน + เส้นอ้างอิง 6 ชม. (จุดสังเกต ไม่ใช่เกณฑ์ตัดสิน)
- [x] กิน: มื้อที่กิน/ข้ามรายวัน + จำนวนเครื่องดื่มหวาน
- [x] เคลื่อนไหว: นาทีเคลื่อนไหวรายวัน แยกสีตามชนิด
- [x] Energy: แถบ low/medium/high รายวัน วางให้เทียบสายตากับกราฟอื่นได้
- [x] วันที่ไม่มีบันทึกแสดงเป็นช่องว่าง ไม่ใช่ศูนย์ (กันกราฟโกหก)

## Acceptance criteria

- ดูกราฟนอน + energy คู่กันแล้ว "เห็น" ความสัมพันธ์ได้ด้วยตาจาก seed data
- ไม่มีการให้คะแนน/เกรดสุขภาพที่ไหนเลย
