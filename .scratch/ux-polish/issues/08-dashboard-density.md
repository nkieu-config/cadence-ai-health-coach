## 1. แท็บกราฟ 4 ปุ่มตัดเป็น 2 แถว — ทุกจอมือถือ ไม่ใช่แค่รุ่นเล็ก

แถบ `ชั่วโมงนอน/การกิน/การเคลื่อนไหว/พลังงาน` ต้องการ ~388px
แต่มือถือให้ได้ 296–398px → ตัดเป็น 2 แถวสูง 104px ในกล่อง `rounded-full`
ที่ออกแบบมาสำหรับแถวเดียว · จอ ≥480 (แท็บเล็ต) ถึงจะพอ

ปุ่มกลุ่มแบบ pill ในแอปมี 3 ที่ ใช้ 3 กลยุทธ์: แท็บกราฟ (ปล่อย wrap — พัง) ·
สลับ 7/14/30 (พอดี) · week picker (เลื่อนได้ เพราะรายการโตได้เรื่อย ๆ)

→ แก้: label สั้น **นอน/กิน/ขยับ/พลังงาน** — "ขยับ" ใช้อยู่แล้วเป็นหัวคอลัมน์ day-lines
ในหน้าเดียวกัน และ legend ใต้กราฟยังคงชื่อเต็ม "ชั่วโมงนอน (ชม.)" ไว้ ข้อมูลไม่หาย
→ บันทึกกฎลง DESIGN.md: รายการคงที่ = จบแถวเดียวที่ 320px · รายการโตได้ = overflow-x-auto

**จงใจไม่แตะ:** ลำดับแท็บ (นอน→กิน→ขยับ) ต่างจาก PILLAR_ORDER (กิน→นอน→ขยับ)
— การนอนคือแกนของ day-lines และมีเส้นอ้างอิง 6 ชม. เปลี่ยน default ก่อนพิตช์ = ถ่าย
screenshot เด็คใหม่อีกรอบ ไม่คุ้ม

## 2. 🔴 marker วันพิเศษหายเงียบเมื่อ recharts ซ่อนแกนวัน

marker ปัจจัยรบกวนถูก render เป็นส่วนหนึ่งของ tick แกน X — พอ recharts ซ่อน tick
ที่แน่นเกินอัตโนมัติ marker ก็หายตามโดยไม่มีสัญญาณอะไรบอก:

| ช่วง | แกนวาดจริง | ผล |
| --- | --- | --- |
| 7 วัน | 7/7 | marker ครบ ✓ |
| **14 วัน** | **8/14 (เฉพาะวันคู่)** | **ศ.17, อ.21 มี deadline จริง (day-lines การ์ดถัดไปแสดงแถวชมพู) แต่กราฟบนไม่มี marker** |
| 30 วัน | 12/26 | หายมากกว่านั้น |

ขัดหลักใน docs/11 ที่ทีมเขียนเองว่า "เราเลือกกราฟที่จริงกว่า แทนกราฟที่ละเอียดกว่า"
— กราฟ 14 วันเล่าเรื่องไม่ครบโดยไม่บอกใคร และเป็น view หลักของ demo

→ แก้: `interval={0}` เมื่อ period ≤ 14 (วาดครบทุกวัน marker ไม่หาย) ·
ลดขนาด marker ที่ 14 วันเป็นไซซ์เดียวกับ 30 วัน + หด foreignObject จาก 44 → 28px
กันพื้นที่กดซ้อนกัน · **30 วันแก้ไม่ได้สะอาดก่อนพิตช์** (26 tick ในกราฟ ~330px)
— บันทึกเป็น known limitation ไว้ทำหลังพิตช์

## 3. อักษรจิ๋วต่ำกว่าเกณฑ์ (ไทยมีสระบน-ล่างซ้อน ต้อง ≥12px)

`text-[10px]` มี 11 จุดทั้งโปรเจกต์ · มองเห็นถาวร 5 จุด → แก้:

- `today-summary.tsx:31` — การกิน/การนอน/เคลื่อนไหว (ไทย 10px การ์ดแรกของหน้า)
- `disruptor-overlay.tsx:181` — เลขวันใต้กราฟ (10px เฉพาะมือถือ ทั้งที่จอใหญ่ได้ 12)
- `day-lines.tsx` — แกนเวลา ("เที่ยงคืน" ไทย 10px → เปลี่ยนเป็น "00:00" สั้นกว่าและ
  ฟอร์แมตเดียวกับ 21:00/03:00 แล้วใช้ 11px แบบตัวเลขล้วน) · "ขยับ" + "x น." → 12px
- `goal-progress-card.tsx:107` — เลขวันที่ในชิป (หน้า /goals)
- `pattern-table.tsx:88` — "ลองทำสัปดาห์นี้" (เดสก์ท็อป)

ปล่อยไว้ (ชั่วคราว/รอง — บันทึกที่นี่): ตัวนับอักษรแชท (โผล่เฉพาะใกล้เต็ม) ·
ข้อความใน popover ปัจจัยรบกวน (เปิดจากการกด)

กติกาใหม่ลง DESIGN.md: **ไทยที่มองเห็นถาวร ≥12px · ตัวเลขล้วนบริบทแน่น ≥11px · ห้าม 10px**
